# BUD-Simple 🌟

子どもの英語発話を後押しするシンプルな Web アプリ

## ✨ 特徴

- 🎤 **簡単音声認識** - ボタン一つで英語録音
- 💬 **温かいフィードバック** - 英語と日本語でサポート
- 📚 **練習履歴** - 今日話した英語を記録
- 💡 **お助けフレーズ** - 困った時のフレーズ表示
- 📱 **モバイル対応** - スマホで気軽に使える

## 🎯 使い方

1. 🎤 ボタンを押す
2. 英語で話す
3. 励ましのメッセージを受け取る
4. また話したくなる！

## 🔧 技術構成

- **フロントエンド**: Vanilla HTML/CSS/JavaScript
- **音声認識**: Web Speech API
- **デプロイ**: Firebase Hosting
- **ストレージ**: localStorage

## 🚀 実行方法

```bash
# ローカル実行
python3 -m http.server 8080

# ブラウザで http://localhost:8080 にアクセス
```
